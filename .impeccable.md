# Design Context

## Users
Single user (Leandro) — a casual crypto hobbyist checking conviction scores daily. Not a professional trader. Uses the dashboard in a relaxed, exploratory manner to inform personal investment decisions. Viewing context: likely evenings or weekends, on desktop, in a calm environment.

## Brand Personality
**Warm, inviting, focused** — like a well-organized personal workshop or craft journal. The interface should feel personal and intentional, not corporate or intimidating. "Deliberately slow" is a core value: daily rhythm, patient analysis, no urgency.

## Aesthetic Direction
- **Tone**: Clean, minimal, modern — inspired by Notion/Linear
- **Theme**: Dark mode (matches evening viewing context)
- **Feel**: Personal tool, not a trading terminal. Warmth over cold precision.
- **Anti-references**: Bloomberg terminals, TradingView density, corporate dashboards

## Design Principles
1. **Warm minimalism**: Clean without being cold. Use warm neutrals, not clinical grays.
2. **Focused attention**: Highlight what matters (action signals), let the rest recede.
3. **Intentional rhythm**: Design reinforces the "deliberately slow" philosophy — no flashing, no urgency.
4. **Personal craft**: Should feel handmade and considered, not template-generated.
5. **Readable at a glance**: Casual user should grasp the state in 2-3 seconds.
